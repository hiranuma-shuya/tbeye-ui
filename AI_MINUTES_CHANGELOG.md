# AI 議事録ページ 仕様変更

## 概要

AI 議事録ページ（ai-minutes.html）の構造と UI を大幅に改善しました。

---

## 1. 時間帯フィルタの追加

右パネル「申し送り記録」に時間帯タブを追加。

| タブ       | 説明                             |
| ---------- | -------------------------------- |
| **朝**     | 朝のアシストコールのみ表示       |
| **昼**     | 昼のアシストコールのみ表示       |
| **夜**     | 夜のアシストコールのみ表示       |
| **すべて** | 全時間帯を表示（デフォルト）     |

---

## 2. 左パネル「記録データ」の変更

### 表示形式

- **旧**: 「文字起こし」という名称で、要約テキストを含む複雑な構造
- **新**: 「記録データ」に名称変更。シンプルなログ形式で表示

```
[15:45:57] [3001] 205号室 佐藤花子さん 心拍が下限設定値を下回ったことをお知らせします。
```

### 折りたたみ機能

- 各アシストコールグループに折りたたみトグル（▼）を追加
- グループヘッダーをクリックで展開/折りたたみを切り替え
- 編集・削除ボタンクリック時はトグルが発動しない

### 編集・削除機能

- 各行に編集(✎)・削除(🗑)ボタンを常時表示
- 親行・子行の両方で編集・削除が可能

---

## 3. 右パネル「申し送り記録」の変更

### 時間帯フィルタ

- 朝・昼・夜・すべてのタブを追加
- タブを切り替えると左パネルの記録データもフィルタリング
- 右パネルの要約も選択した時間帯に対応

### フッター

- 「再要約」ボタンを追加（記録データ編集後に要約を再生成）

---

## 4. ヘッダー UI の整理

以下の要素をヘッダーに集約：

- フォーマット選択（メール用/保存用）
- 「辞書に追加」ボタン
- 辞書 ON/OFF トグル（新デザイン）

## 削除した機能

- 「申し送り/一日の終わり」モード切り替え（申し送りモードのみに統一）
- 「要約生成」ボタン（要約は自動生成のため不要）
- 履歴タブ（シンプル化のため削除）
